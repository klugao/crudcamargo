# Sistema de Venda de Ingressos

## Sistemas implementados no site:
- **Login**
- **Cadastro**
- **Compra**
- **Painel de Administração:** Permite adicionar e remover eventos.
- **Sistema de Token:** Para validação.
- **API do Firebase**
- **Sistema de Geo-localização**
